# 🎉 College Fest Landing Page

A modern, responsive, and visually appealing **College Fest Landing Page** built using **HTML5, CSS3, and JavaScript**. The website is designed to showcase a college technical festival with attractive UI, smooth animations, and a user-friendly experience.
